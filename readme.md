## Creacion del entorno Virtual

1:

``` bash
python -m venv venv
```

or

``` bash
python3 -m venv venv
```

2:

``` bash
# Linux
source .venv/bin/activate

#W11
gad-env\Scripts\activate
```

Instalar las dependencias (Solo la primera vez)

3:

```bash
pip install -r requirements.txt
```
